# PubIt - Obsidian Plugin

Publish your Obsidian notes to the web with zero friction.

## Features

- Publish notes to `yourname.pubit.site` with one click
- Tag notes with `#publish` or set `publish: true` in frontmatter
- Automatic sync - only changed notes are uploaded
- Wikilinks work across published notes
- Image support included

## Installation

### From Obsidian Community Plugins (Coming Soon)

1. Open Obsidian Settings
2. Go to Community Plugins and disable Safe Mode
3. Click Browse and search for "PubIt"
4. Install and enable the plugin

### Manual Installation

1. Download `main.js` and `manifest.json` from the [latest release](https://github.com/jacoblindqvist/obsidian-pubit/releases/latest)
2. Create a folder called `pubit-publish` in your vault's `.obsidian/plugins/` directory
3. Copy `main.js` and `manifest.json` into that folder
4. Reload Obsidian and enable the plugin in Settings > Community Plugins

## Usage

1. Open the plugin settings and sign in with your account
2. Choose or create your subdomain (e.g., `yourname.pubit.site`)
3. Add `#publish` tag to any note you want to publish
4. Click the publish icon in the ribbon or run the "Publish notes" command
